What I learned about HTML so far.

  • A website can be made using HTML on basically any computer, with just built in software.
  • HTML is a markup language, not a programming language.
  • HTML uses tags to tell a website how to display text and images.
  • HTML can be used with CSS to change the style of the website.
  • HTML is very versatile, so websites can look very different from one another.
  • HTML uses hyperlinks to get the user from one web page to another.
  • A coherent collection of web pages is a website.
  • W3Schools is a great website to learn how to use HTML and CSS.
